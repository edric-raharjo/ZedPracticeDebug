# Important Zed Shortcuts

## 1. Go to Definition

- Go to a function name, then press `F12` to jump to its definition.

---

## 2. Project Panel

Press `Ctrl + Shift + E` to switch to the Project Panel.

### Navigation

- Use **Up** and **Down** to navigate.
- Use **Left** to collapse and **Right** to expand.
- `Ctrl + Left` collapses all.
- `Ctrl + Right` doesn't exist.
- Press **Spacebar** to open a file temporarily.
- Start editing the file to permanently open it. (Double clicking with a mouse can work too)

---

## 3. Search in Project

Press `Ctrl + Shift + F` to search for a string in the entire project.

---

## 4. Project Symbols

Press `Ctrl + Shift + B`.

> `Ctrl + Shift + O` also works, but that's more like a `Ctrl + P` equivalent.

This lets you search through project structure:

- Classes
- Functions
- Methods
- Other symbols

## 6. Multi-buffer Guide

1. Select the function name you want to apply multi-buffer to.
2. Press `Ctrl + Shift + F`, then press `Enter`.
3. To select all occurrences of the function name, press `Ctrl + Shift + L`.
4. Use the arrow keys to navigate between the multi-buffer results.
5. Press `Alt + Enter` to jump to the file containing the selected occurrence.
